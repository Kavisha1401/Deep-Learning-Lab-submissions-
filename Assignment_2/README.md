## Problem Statement

This assignment focuses on building and evaluating **Convolutional Neural Networks (CNNs)** for handwritten digit classification using the **MNIST dataset**. The goal is to analyze how different architectural and training choices affect model performance.

## Tasks

- Build a CNN for MNIST classification
- Experiment with different:
  - Activation functions
  - Optimizers
  - Learning rates
  - Batch Normalization
  - Dropout
- Train and evaluate models using classification accuracy

## Objectives

- Understand the impact of model design choices
- Compare training stability and performance across configurations
- Gain hands-on experience with CNNs in TensorFlow/Keras

## Tools Used

- Python  
- TensorFlow / Keras  
- NumPy  
- Matplotlib

## Expected Outcome

- CNN achieves high accuracy on MNIST
- Batch Normalization and Dropout improve training stability
- Optimizer and activation choice significantly affect convergence
